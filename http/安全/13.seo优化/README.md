## SEO优化方案
1. 合理利用`title`, `description`, `keywords`
> 搜索对着三项的权重逐渐减少，title强调重点即可，description把页面高度概括，keywords列举重要的关键词
2. 语义化HTML代码，符合W3C规范
> 语义化代码让搜索引擎容易理解网页
3. 重要的内容的HTML放在最前面
> 搜索引擎抓取HTML是顺序的从上到下， 保证重要内容一定会被抓取
4. 重要的内容不要JS输出
> 爬虫不会执行JS的内容
5. 少用iframe
> 搜索引擎不会抓取iframe的内容
6. 提高网页的速度
> 网站的速度是搜索引擎的一个重要指标
